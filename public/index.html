About Me

A 27-year-old Linux enthusiast and Head of the IT Security and Administration Department, I am committed to advancing cybersecurity measures and streamlining IT infrastructure. My role focuses on developing strategic solutions that enhance data protection and improve system reliability across all operational levels.

  • Bash Scripting

    Developing advanced Bash scripts, I enhance system efficiency by automating repetitive tasks and streamlining data processing and system integration workflows. These scripts optimize operations across various platforms, ensuring seamless interactions and rapid deployment of system updates.

  • Server Monitoring

    I oversee server monitoring with Zabbix for optimal performance and stability. This involves managing server health and performance metrics for swift issue resolution. Alerts quickly notify the IT team of critical issues, ensuring rapid responses. Regular reviews with Zabbix data facilitate resource scaling to support business growth without compromising security.

  • Data Security Management

    I oversee data security strategies, including encryption, access controls, and comprehensive security policies to protect sensitive information. Additionally, I design and manage robust backup procedures that ensure data resilience and continuity in the event of data loss, providing a secure framework for data recovery.

  • Ansible Automation

    Utilizing Ansible, I focus on automating configuration, management, and deployment of applications across servers. My work includes creating playbooks that standardize environments and reduce manual intervention, enhancing consistency and reliability in server setups and software deployments.

  • Server Maintenance

    I manage server infrastructure maintenance, ensuring high availability and reliability of physical, virtual, and cloud servers essential for business operations. My role includes routine updates, security patches, kernel tuning, network configuration, and system optimization to boost performance. I oversee multiple Windows and Linux servers, ensuring they are secure, up-to-date, and configured according to best practices.

  • Networking

    I manage network using FortiGate tools, enhancing security through a suite of features. Firewall monitors all network traffic to prevent cyber threats, while the Intrusion Prevention System proactively stops attacks based on known patterns. I also oversee Virtual Private Networking (VPN) for secure remote access and implement Web Filtering to control internet usage according to organizational policies.
